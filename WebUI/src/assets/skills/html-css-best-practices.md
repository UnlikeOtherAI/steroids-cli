# HTML & CSS Best Practices
1. Use semantic HTML tags (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`).
2. Avoid div-soup. Only use `<div>` when no other semantic tag makes sense.
3. Keep CSS Vanilla where possible. Do not use Tailwind unless the project is already heavily configured for it.
4. Use flexbox or grid for layout; avoid floats.
5. Provide accessibility tags (aria-labels, alt text for images).
