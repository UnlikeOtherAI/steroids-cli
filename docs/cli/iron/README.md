# Iron

> Documentation scaffolding CLI - set up project documentation the right way.

---

## What is Iron?

Iron is a CLI tool that guides you through setting up initial documentation for your projects. It scaffolds the essential docs structure so you can focus on building, not boilerplate.

**Part of the Steroids suite:** Pump Iron + Steroids

---

## Planned Features

- Interactive documentation setup wizard
- Templates for common project types (Node, Python, Go, etc.)
- CLAUDE.md / AGENTS.md generation for LLM-assisted development
- README scaffolding with sensible defaults
- API documentation stubs
- Architecture decision records (ADRs)
- Changelog initialization
